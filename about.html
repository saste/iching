About this website

This website was done as an assignment for the course in "Web development", kept at the University of Cagliari by Fabio Sorrentino, for the Bachelor Degree Course in Computer Science.

I had to learn and use a lot of cool stuff in order to get the site working, in particular I used:

  • PHP for what regards the server-side processing.
  • JQuery - the most coolest Javascript library ever written, and the JQuery form plugin.
  • MySQL as server-side database server.
  • The One True Editor for all the typing/editing operations.

The idea of the website comes from the Iching Online website, which I consider pretty good from the design point of view, and this implementation basically borns as a barebone port of the functionality provided by that site, but without the ads!

That was a painful process to learn how to tame and bind together all that stuff, but I had some fun and possibly I learnt something along the way.

Enjoy!
